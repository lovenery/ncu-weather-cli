# ncu-weather-cli [![npm package](https://img.shields.io/npm/v/ncu.svg)](https://www.npmjs.com/package/ncu)
A simple CLI for NCU's Information

### Requirements
- Node.js 6.x

### Install
```bash
$ npm install ncu -g
```

### Usage
```bash 
$ ncu
```
